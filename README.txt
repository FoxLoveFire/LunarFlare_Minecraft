#LunarFlare in Minecraft


Based on Minecraft Forge 1.12.2


#Installation 

Download [IntelliJ Idea](https://www.jetbrains.com/idea/download/?section=windows) Recommended version: 2021.1


Download Java 8 and JDK 

Open project in IntelliJ Idea. Build project. 

In terminal in the dir of project run:

Windows:
```
gradlew.bat genIntellijRuns
```

Linux:

```
./gradlew.bat genIntellijRuns
```

In IntelliJ Idea. Open "Application" button. Choose "run client".
